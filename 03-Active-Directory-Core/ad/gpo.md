\### RDP Access Control Policy

\*\*GPO Name:\*\* `GPO\_Restringir\_RDP`  

\*\*Path:\*\* `Computer Configuration > Policies > Windows Settings > Security Settings > Local Policies > User Rights Assignment`  

\*\*Setting:\*\* `Allow log on through Remote Desktop Services`  

\*\*Logic:\*\* Explicitly allowed only the `Administrators` group to reduce the attack surface for RDP brute-force attempts.
